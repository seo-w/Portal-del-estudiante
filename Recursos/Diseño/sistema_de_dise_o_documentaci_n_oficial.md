# 🎓 Sistema de Diseño — Academia Ecommerce 24/7
### *"Donde Udacity se encuentra con Stripe"* — Guía de Identidad Visual y UI

---

## 🧬 ADN de la Marca

La plataforma fusiona la robustez de un sistema educativo de clase mundial (**Udacity**) con el acabado visual sofisticado y tecnológico de una fintech de élite (**Stripe**). El resultado es una interfaz que transmite confianza, claridad y modernidad.

- **Legibilidad:** Predominancia de espacios en blanco y tipografía navy para una jerarquía clara.
- **Atmósfera:** Uso controlado de gradientes "mesh" y sombras quirúrgicas para elevar la percepción de calidad.
- **Acción:** Un color índigo vibrante guía al usuario a través de su proceso de aprendizaje.

---

## 🎨 Paleta de Colores

### Fondos y Superficies
- **Página Principal:** `#FFFFFF` (Blanco puro para máxima claridad)
- **Superficies Secundarias:** `#F8F9FB` (Gris azulado muy suave para sidebars y cards)
- **Subtle Surface:** `#F1F3F7` (Hovers y elementos de fondo neutros)
- **Bordes:** `#E2E8F0` (Líneas finas y precisas)

### Colores de Marca (Acentos)
- **Índigo Primario:** `#4F46E5` (Color de acción, botones y estados activos)
- **Cyan Secundario:** `#0EA5E9` (Highlights, iconos de herramientas y progreso)
- **Violeta Decorativo:** `#7C3AED` (Utilizado en gradientes mesh y estados especiales)
- **Éxito (Emerald):** `#10B981` (Validaciones, completado y métricas positivas)

### Gradientes Especiales
- **Mesh Gradient Hero:** Radial gradients de índigo, violeta y cyan con opacidad < 10% sobre fondo blanco.
- **Botón Primario:** `linear-gradient(135deg, #5B21B6, #4F46E5, #0EA5E9)`
- **Texto Hero:** Gradiente desde `#7C3AED` hasta `#0EA5E9` para títulos de impacto.

## 🔤 Tipografía

### Estilo de Fuente
- **Headings (Títulos):** `Plus Jakarta Sans` (Sans-serif moderna, geométrica y con excelente legibilidad en negrita).
- **Body (Cuerpo):** `DM Sans` (Limpia y funcional para textos largos y controles de UI).
- **Mono:** `JetBrains Mono` (Utilizada para datos métricos, precios y códigos).

### Escala Tipográfica
- **H1 Hero:** 60px / 800 (Bold) / -0.04em tracking
- **H1 Página:** 42px / 800 / -0.03em tracking
- **H2 Sección:** 32px / 700 / -0.02em tracking
- **Cuerpo Base:** 16px / 400 (Regular) / 1.6 line-height
- **Labels UI:** 13px / 500 / 0.06em uppercase

## 📐 Layout y Componentes

### Grid y Espaciado
- **Contenedor Máximo:** 1280px.
- **Sidebar Portal:** 260px fijo con fondo `#F8F9FB`.
- **Gaps Estándar:** 24px para grids de herramientas y 20px para cards de cursos.

### Componentes Clave
1. **Sidebar de Navegación:**
   - Estado Activo: Fondo blanco, borde izquierdo de 3px índigo y sombra sutil.
   - Iconografía: Estilo "Outline" con trazo refinado.
2. **Course Cards:**
   - Estructura Udacity: Thumbnail superior, progress bar inferior.
   - Efecto Hover: Elevación sutil (`translateY(-2px)`) y refuerzo de sombra.
3. **Tool Cards:**
   - Más grandes y prominentes.
   - Incluyen lista de features con checks de colores y CTAs específicos.
4. **Formularios (Stripe-style):**
   - Inputs con bordes redondeados (8px) y focus ring de alta visibilidad en índigo.
